Machine Learning


We performed a random forests on our initial feature set of 71 columns and were able to reduce it down to the top ten columns reference in the figure. These ten features are:

Together, these ten features explained about 64% of the variation of the target variable: which was whether the player made the AllStar game or not. We deployed these ten features in five separate supervised machine learning models and they had accuracy and recall rates of about 91% and 92% for all models.

1.) Points per Game
2.) Total Points
3.) Field Goals
4.) Free Throws
5.) Free Throws Attempted
6.) Field Goals Attempted
7.) Minutes per Game
8.) Value Over Replacement Player
9.) Win Shares
10.) Two Pointers Made
